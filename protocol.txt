1 - El croupier envia un missatge amb id 100 (què teniu a la mà?) 
2 - Els jugadors contesten amb id 100 + n (on n és el número de grup) indicant què tenen a la mà. Cada grup tindrà una clau amb la qual encriptarà aquest missatge
3 - El croupier envia un missatge amb id 200 (feu les apostes) 
4 - Els jugadors contesten amb id 200 + n (on n és el número de grup) indicat la seva primer aposta
5 - El croupier torna a enviar un missatge amb id 200 (torneu a fer les apostes)
6 - Els jugadors contesten amb id 200 + n (on n és el número de grup) indicat la seva aposta final
7 - El croupier envia un missatge amb id 300 indicant qui ha estat el guanyador

- Es torna a repetir la seqüència
